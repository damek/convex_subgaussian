\chapter{Overview}\label{chap:overview}

This blueprint maps the Lean formalization of the sharp one-dimensional
Gaussian convex comparison theorem. The public endpoint proves that the
smallest Gaussian scale dominating every centered real random variable with
two-sided sub-Gaussian tails is the explicit constant $c_0$, and that every
smaller scale fails against a canonical extremal law.

Every theorem cited here is formalized in Lean without \texttt{sorry} and the
reader-facing endpoint is intentionally small: the file
\texttt{ConvexSubgaussian/GaussianMain.lean} exposes only the main scale
definition and the final sharp theorem.
